### My Journey So Far

I started my programming journey by experimenting with fullstack web development using React and Java.
It began as a hobby, creating simple yet functional websites and gradually expanding my skills in web development.

Over time, my interests shifted towards deeper aspects of software engineering. I became curious about systems-level programming and decided to explore Rust, known for its performance and safety features.

Transitioning from web development to systems programming with Rust has been an interesting shift. While the learning curve was steep, I found the experience to be both challenging and rewarding.

Today, I appreciate the versatility of programming, from crafting user interfaces to building robust systems. Each step in my journey has been an opportunity to expand my skills and explore new technologies.
